**JavaScript**

## 복제방법
```
$ git clone https://github.com/Pigbot-official/opensource

$ cd opensource

$ npm i --save

$ node index.js
```
## 참고사항
* **복제후 config/bot.json 파일에 token 부분에 봇 토큰을 넣어주세요**
